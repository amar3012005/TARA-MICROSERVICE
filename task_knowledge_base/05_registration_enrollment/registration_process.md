# T.A.S.K Registration Process - Complete Guide

**Document Type**: Registration Procedure  
**Last Updated**: December 2025  
**Purpose**: Step-by-step registration guide for RAG retrieval

---

## Overview

T.A.S.K registration is a 10-step online process that takes approximately 15-30 minutes to complete. Most information can be saved and completed later.

---

## Step-by-Step Registration Process

### Step 1: Portal Access
**Action**: Visit T.A.S.K official website
- **URL**: https://www.task.telangana.gov.in
- **Click**: "Register" or "Student Login" button
- **Note**: Ensure stable internet connection

### Step 2: Account Creation
**Action**: Create your T.A.S.K account
- **Email**: Enter valid email address (will be used for all communications)
- **Password**: Create strong password (minimum 8 characters, mix of letters, numbers, symbols)
- **Phone Verification**: Enter mobile number and verify via OTP
- **Confirmation**: Check email for verification link

**Important**: 
- Use active email address (check spam folder)
- Keep phone number accessible for OTP
- Remember your password or save it securely

### Step 3: Personal Information
**Action**: Fill personal details
- **Name**: Full name as per official documents
- **Date of Birth**: DD/MM/YYYY format
- **Gender**: Select from options
- **Address**: Complete address with PIN code
- **Current Location**: City and district
- **Employment Status**: Student/Unemployed/Working Professional

**Note**: All fields marked with * are mandatory

### Step 4: Academic Information
**Action**: Enter academic details

**For Engineering/Polytechnic Students**:
- **College Name**: Select from dropdown or enter manually
- **Program**: Engineering/Diploma
- **Branch**: CS, ECE, EEE, Mechanical, Civil, etc.
- **Current Year**: 1st, 2nd, 3rd, 4th year
- **Aggregate Percentage**: Enter percentage (minimum 50% required)
- **College ID Number**: Enter college ID

**For Graduates**:
- **Degree**: Bachelor's degree name
- **University**: University name
- **Year of Graduation**: Year completed
- **10th Percentage**: Minimum 60% required
- **Intermediate Percentage**: Minimum 60% required
- **Graduation Percentage**: Minimum 60% required

**For Faculty**:
- **Institution**: Teaching institution name
- **Department**: Department name
- **Experience**: Years of teaching experience
- **Qualification**: Highest qualification

### Step 5: Program Selection
**Action**: Choose your programs
- **Primary Interest**: Select your main program interest
  - AI & Machine Learning
  - Cloud Computing (AWS/Azure/GCP)
  - Cybersecurity
  - Data Science
  - Full Stack Development
  - Java Programming
  - Manufacturing & Quality
  - Healthcare
  - Others

- **Secondary Interests**: Select up to 3 additional programs
- **Preferred Batch Timing**: Morning/Afternoon/Evening/Weekend
- **Preferred Format**: Online/Offline/Hybrid

**Note**: You can change program selection later, but primary interest determines initial placement

### Step 6: Placement Preferences
**Action**: Set your career preferences
- **Preferred Sector**: IT, Manufacturing, Healthcare, Data Science, etc.
- **Salary Expectation**: Range (e.g., ₹3-5 LPA)
- **Preferred Location**: Hyderabad, Bangalore, Pune, Remote, etc.
- **Job Type**: Full-time, Internship, Contract, etc.

**Note**: These preferences help T.A.S.K match you with relevant opportunities

### Step 7: Document Upload
**Action**: Upload required documents

**Required Documents**:
- **College ID**: Scanned copy or photo (for students)
- **Aadhar Card**: Scanned copy or photo
- **Photo**: Recent passport-size photo
- **Academic Records**: Marksheets (for graduates)

**Document Requirements**:
- **Format**: PDF, JPG, PNG (max 2MB per file)
- **Quality**: Clear and readable
- **Validity**: Documents should be valid and current

**Upload Process**:
1. Click "Upload" button
2. Select file from device
3. Wait for upload confirmation
4. Verify uploaded document preview
5. Repeat for all documents

### Step 8: Review & Agreement
**Action**: Review all information and accept terms
- **Review**: Check all entered information for accuracy
- **Edit**: Click "Edit" to modify any section
- **Terms & Conditions**: Read and accept T.A.S.K terms and conditions
- **DPDP Compliance**: Accept data privacy and protection terms
- **Consent**: Provide consent for data usage and communication

**Important**: 
- Ensure all information is accurate
- Read terms and conditions carefully
- Contact support if you have questions

### Step 9: Payment (if applicable)
**Action**: Complete payment for paid programs

**Payment Methods**:
- **UPI**: Google Pay, PhonePe, Paytm
- **NetBanking**: All major banks
- **Debit Card**: Visa, Mastercard, RuPay
- **Credit Card**: Visa, Mastercard
- **NEFT/RTGS**: Bank transfer
- **Cheque**: For organizations

**Payment Process**:
1. Select payment method
2. Enter payment details
3. Review payment amount
4. Complete transaction
5. Save payment receipt

**Note**: 
- Most programs are free or heavily subsidized
- Payment is only required for premium programs
- Scholarships and financial assistance available

### Step 10: Confirmation
**Action**: Receive confirmation and next steps

**Confirmation Details**:
- **Email Confirmation**: Check email for confirmation message
- **Enrollment ID**: Unique enrollment ID (save this)
- **Portal Access**: Login credentials for student portal
- **Next Steps**: Orientation date, program start date, contact information

**What Happens Next**:
- **Immediate (24 hours)**: Portal access, LMS resources available
- **Within 48 hours**: Orientation email, assessment details, peer group assignment
- **Week 1**: Program kickoff, industry partner introduction, course materials

## Common Questions Answered

**Q: How long does registration take?**  
A: Registration takes 15-30 minutes if you have all documents ready. You can save progress and complete later.

**Q: What if I make a mistake during registration?**  
A: You can edit information before final submission. After submission, contact registrations_task@telangana.gov.in for corrections.

**Q: Can I register for multiple programs?**  
A: Yes, you can select primary interest and up to 3 secondary interests during registration.

**Q: Is payment required immediately?**  
A: Payment is only required for premium programs. Most programs are free or heavily subsidized. You can complete payment later if needed.

**Q: What if I don't receive confirmation email?**  
A: Check spam folder, verify email address, or contact registrations_task@telangana.gov.in for assistance.
