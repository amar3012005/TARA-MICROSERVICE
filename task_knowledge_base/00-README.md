# T.A.S.K Knowledge Base

## Overview

This knowledge base is designed for the T.A.S.K (Telangana Academy for Skill and Knowledge) customer service voice agent. It contains comprehensive information organized into **12 categories**, covering every aspect of T.A.S.K operations, programs, student services, and organizational information.

## Purpose

This knowledge base is optimized for:
- **RAG (Retrieval Augmented Generation)** pipelines
- **Ultra-low latency** response generation
- **Fast contextual** information retrieval
- **Accurate** customer service responses
- **Telugu-English mixed** language support
